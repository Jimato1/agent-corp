**PrintedAbsence** — a calm locked fact for a capability that can't exist by construction; never a greyed-out toggle.

```jsx
<PrintedAbsence>
  <strong>An agent can never approve its own work.</strong> Approval requires a different principal.
</PrintedAbsence>

<PrintedAbsence glyph="⛊" why="Rotate or revoke it instead." tag="never readable">
  The vault never displays a stored secret back.
</PrintedAbsence>
```

- Use where the power genuinely cannot be switched on — reassures by explaining the absence.
- Never use ⛔ here (that glyph is only for actionable stops). Not interactive.
